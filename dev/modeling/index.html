Modeling the text collection

The processesing and tokenization is defined through a text configuration TextConfig.

Once a TextConfig is define, we need to create

We need to create a model for the text, we select a typical vector model. The model constructor needs to know the weighthing scheme and some stats about the corpus' vocabulary:

julia> corpus_bow = compute_bow(config, corpus)

julia> modelfreq = VectorModel(FreqWeighting(), corpus_bow)
{VectorModel weighthing=FreqWeighting(), vocsize=45659, maxfreq=22243}

julia> modeltf = VectorModel(TfWeighting(), corpus_bow)
{VectorModel weighthing=TfWeighting(), vocsize=45659, maxfreq=22243}

julia> modelidf = VectorModel(IdfWeighting(), corpus_bow)
{VectorModel weighthing=IdfWeighting(), vocsize=45659, maxfreq=22243}

julia> modeltfidf = VectorModel(TfidfWeighting(), corpus_bow)
{VectorModel weighthing=TfidfWeighting(), vocsize=45659, maxfreq=22243}

Now we can vectorize a text

julia> text = "las mejor música, la música de siempre!"
"las mejor música, la música de siempre!"
